Stovetop
========

An empty HTML5 and CSS3 starting plate for a quick start to simple web projects.


## What's in it

Stovetop contains the following files:

```
css/normalize.css
css/style.css
index.html
```


## How to use it

Just download it and add your own code. Pretty simple.
